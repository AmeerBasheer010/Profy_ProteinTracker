Indian Protein Tracker App

A simple, protein tracker built specifically 
for Indian users. No login. No paywall. Just track 
your daily protein intake easily.

---

Features
✅ Clean and simple UI
✅ 100+ Indian foods database
✅ Kerala & South Indian foods included
✅ Shakes, nuts, supplements included
✅ Add food with quantity selector
✅ Delete food with confirmation popup
✅ Daily protein progress tracker
